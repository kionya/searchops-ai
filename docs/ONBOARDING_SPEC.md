# Onboarding Flow Draft

Initial customer onboarding should guide a user through:

1. Create or join an organization.
2. Add a site domain.
3. Confirm tenant and role context.
4. Run a first crawl.
5. Review SEO issues.
6. Create the first work order.
7. Recheck after a CMS/content update.

The flow should remain safe without live connectors. GSC, GA4, PageSpeed, Bing, CMS, GEO provider, and billing setup can be shown as optional readiness steps until credentials are provisioned.
