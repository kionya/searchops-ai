# Privacy Policy Draft

SearchOps AI stores site, crawl, connector, SEO issue, work order, and operational metadata needed to run SEO/AEO/GEO diagnosis and rechecks.

No secrets, API tokens, raw provider credentials, or real customer payloads should be committed to the repository or fixture files. Production credentials must live in deployment secret storage.

Medical and compliance-related content is processed as draft/review material only. The platform does not auto-publish medical content.

This draft must be reviewed by legal counsel before production launch.
